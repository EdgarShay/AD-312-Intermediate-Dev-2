# Shopping List with useImmer
Author: Edgar

## Description
A React application that demonstrates complex state management using the useImmer hook from Immer.

## Features
- Add shopping items
- Update item details
- Remove items
- Nested state handling

## Technologies
- React
- Vite
- Immer (useImmer)

## How to run
npm install
npm run dev
http://localhost:5173/
