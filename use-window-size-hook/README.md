# useWindowSize Custom Hook
Author: Edgar
Packaging Reusable Logic with Custom Hooks

## Overview

This project demonstrates how to create and use a reusable React custom hook called useWindowSize.

The hook tracks the browser window's width and height and updates automatically whenever the window is resized.

## Features

* Custom React Hook
* useEffect for side effects
* Event listener cleanup
* Real-time window dimensions
* Mobile and Desktop layouts

## Installation

npm install

## Run Project

npm run dev
