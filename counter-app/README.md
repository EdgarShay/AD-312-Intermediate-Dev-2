# React Counter App

## Description
This project demonstrates React state management using the useState hook.
It shows how React batches updates and how state behaves like a snapshot.

## Features
- Increment counter
- Increment after delay (async)
- Incorrect double update (batching example)
- Correct double update using functional state

## Technologies Used
- React
- Vite
- JavaScript (ES6)

## How to Run
1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

## Concepts Demonstrated
- useState Hook
- Asynchronous state updates
- React batching
- State as a snapshot
