# Dynamic Poll Dashboard

Author: Edgar

## Overview

This project demonstrates how to integrate a non-React JavaScript library (Chart.js) into a React application using useEffect and useRef.

The application allows users to vote for their favorite JavaScript framework and updates a Chart.js bar chart in real time.

## Technologies

* React
* Vite
* Chart.js
* Vitest
* React Testing Library

## Features

* Vote for React, Vue, or Angular
* Real-time chart updates
* Chart.js integration using useEffect
* Proper cleanup using destroy()
* Unit testing with Vitest

## Installation

```bash
npm install
npm run dev
```

## Running Tests

```bash
npm run test
```

## Learning Objectives

* React useEffect lifecycle
* useRef for external library instances
* Integrating non-React libraries
* Preventing memory leaks
* Component testing
