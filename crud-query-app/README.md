# CRUD Query App
Author: Edgar

## Overview
This project demonstrates CRUD operations using TanStack Query and JSONPlaceholder in a React Native Expo application.

## Features
- Fetch posts
- Create posts
- Patch posts
- Delete posts
- Filter posts by User ID
- Loading and error handling

## Technologies
- React Native
- Expo
- TanStack Query
- JSONPlaceholder API

## Installation

## bash
npm install
npx expo start
