# useLocalStorage Custom Hook
## Author: Edgar

Student Assignment: Syncing State with LocalStorage Using a Custom Hook

## Overview

This project demonstrates a reusable React custom hook called useLocalStorage.

The hook combines React state with browser localStorage to automatically save and restore values.

## Features

* Custom Hook
* useState Initialization Function
* localStorage Persistence
* useEffect Synchronization
* Theme Toggle Example

## Installation

npm install

## Run

npm run dev

## Technologies

* React
* Vite
* JavaScript
* CSS
