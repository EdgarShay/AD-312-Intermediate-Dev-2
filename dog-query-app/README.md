# Dog Query App 🐶

Author : Edgar

## Overview
This project uses TanStack Query to fetch and manage data from the Dog API.

## Features
- Fetch and display dog breeds
- View detailed breed information
- Display dog facts
- Show dog groups
- Handle loading, error, and success states

## Technologies
- React (Vite)
- TanStack Query
- JavaScript
- CSS

## Installation

```bash
npm install
npm run dev

Data fetching with TanStack Query
