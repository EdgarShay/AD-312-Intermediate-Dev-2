# Responsive Card Using useEffect
Author: Edgar

## Overview

This React application demonstrates the use of the useEffect hook to synchronize a component with the browser window size.

The application:

* Tracks browser width and height in real time
* Updates automatically when the browser is resized
* Changes between Mobile and Desktop layouts
* Uses event listeners correctly
* Includes cleanup logic to prevent memory leaks

## Technologies

* React
* Vite
* JavaScript
* CSS

## Running the Project

Install dependencies:

npm install

Start development server:

npm run dev

Open the provided localhost URL in your browser.

## Features

* Real-time window size tracking
* Responsive layout detection
* Dynamic styling
* Proper useEffect cleanup
* Mobile and Desktop views
