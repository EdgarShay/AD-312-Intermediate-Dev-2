# 🚀 User Registration Form (React Hook Form)
Written by: Edgar

A modern, high-performance user registration form built with **React Hook Form**.
This project demonstrates advanced form handling techniques including validation, async submission states, draft caching, and optimized rendering.

---

## Features
- React Hook Form (uncontrolled inputs)
- Real-time validation
- Password strength rules
- Confirm password matching
- Role selection dropdown
- Terms & conditions validation
- Auto-focus on load
- Draft saving with localStorage
- Async submission simulation (2 seconds)
- Loading state handling
- Form reset after success

---

## Tech Stack

- React (Vite)
- React Hook Form
- JavaScript (ES6+)
- CSS3 (custom styling)
