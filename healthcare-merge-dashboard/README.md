# Healthcare Merge Dashboard
Written by: Edgar
## Overview
This project merges patient healthcare records from two providers using sorted linked lists.

## Technologies
- React
- Node.js
- Express
- Jest
- CSS

## Features
- Merge sorted linked lists
- Search patients by SSN
- Add patient form
- REST API
- Unit testing
- SaaS dashboard UI

## Run Backend
cd backend
node server.js

## Run Frontend
cd frontend
npm run dev

## Run Tests
cd backend
npm test

## Time Complexity
O(n + m)

## Space Complexity
O(1)
